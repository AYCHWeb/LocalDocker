## Description

Local docker registry cache

## Requirements

  * Linux (tested on Ubuntu 16.04)
  * Docker (version 17 or higher)
  * Docker-compose (version 1.18 or higher)
  * ~/.docker/config.json with creds to source repository

## Installation

Just run ./setup.sh
